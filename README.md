# Calculatrice Simple en Python

## Description

Cette application est une calculatrice graphique développée en Python avec la bibliothèque Tkinter. Elle permet d'effectuer des opérations mathématiques de base à travers une interface simple et intuitive.

## Fonctionnalités

* Addition (+)
* Soustraction (-)
* Multiplication (*)
* Division (/)
* Modulo (%)
* Nombres décimaux
* Effacement de l'expression (C)
* Calcul du résultat (=)

## Technologies utilisées

* Python 3
* Tkinter

## Installation

1. Cloner le dépôt :

```bash
git clone https://github.com/votre-utilisateur/calculatrice-python.git
```

2. Accéder au dossier du projet :

```bash
cd calculatrice-python
```

3. Exécuter l'application :

```bash
python3 calculatrice.py
```
